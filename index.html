Blog Image Gallery

This is an image gallery I made for a Blog site. It was created using PHP, Javascript, HTML, CSS and MySQL. The image gallery allows the user to upload their photos to show to friends or followers in a clean, simple format. In order to make the upload process easier I utilised the Javascript plugin 'Dropzone' which allows the user to simply drag and drop multiple files to be uploaded at the same time. In addition the gallery also includes an admin panel which allows the user to edit or delete their pictures. The admin dashboard also includes a Google Chart to visualise multiple site statistics such as number of views, number of pictures in the database and how many comments each picture has etc.
